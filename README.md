# devroast

> Cole seu código. Receba um roast brutal.

**devroast** é um app onde você cola qualquer trecho de código e recebe uma avaliação implacável gerada por IA — seja ela honesta demais ou no modo roast completo, com sarcasmo máximo ativado.

Construído ao vivo durante o evento **NLW (Next Level Week)** da [Rocketseat](https://rocketseat.com.br), nas aulas do evento.

---

## O que o app faz

- **Editor de código** — cole ou digite seu código diretamente no navegador
- **Roast Mode** — ative para receber uma avaliação com sarcasmo máximo
- **Score de vergonha** — seu código recebe uma nota de 1 a 10 (quanto menor, pior)
- **Leaderboard** — ranking público com os piores códigos já avaliados, ordenados pela nota

---

## Tecnologias

- [Next.js 16](https://nextjs.org) com App Router
- [React 19](https://react.dev)
- [Tailwind CSS v4](https://tailwindcss.com)
- [Shiki](https://shiki.matsu.io) para syntax highlighting
- [Base UI](https://base-ui.com) para componentes acessíveis
